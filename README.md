## What this repository is

This is a **Blueprint Repository** — the home for how we think, plan, and build.

Each **top-level folder is one concept**: a project, a platform, a design, or an initiative. Every concept manages its own documentation internally (subfolders for diagrams, notes, designs, and plans as needed) and exposes **one entry document** that gives a complete high-level overview on its own.

The homepage you are reading stays deliberately small: it only tells you what exists and where to start reading.

## Adding a new concept

1. Create one top-level folder named after the concept (`kebab-case`).
2. Give it a `README.md` that works as a **complete, self-contained overview**.
3. Put supporting material (diagrams, notes, designs, plans) in subfolders inside the concept.
4. Add one row to the Concepts table above.

That's the entire structure. Keep the root clean.
